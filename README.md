# cornell-msba-datafest-2026
Official dataset and materials for Cornell MSBA Datafest 2026

Welcome to the official dataset repository for the Cornell MSBA DataFest 2026.
Teams will analyze the dataset and develop a creative, data-driven solution.

Download the dataset here:
👉 https://drive.google.com/file/d/1hEXu9A2zSPSFXTC2GpuOzFEj_Z51dr6p/view?usp=sharing

Important:
Download and unzip the file locally before analysis
Do not modify the original dataset file

Challenge Overview: 
Over a three-week period, teams will build a solution using the dataset.
Your final output may take the form of:
  A dynamic dashboard
  A predictive model
  A strategic slide deck

Research Prompt:
Using the provided dataset, identify meaningful patterns, insights, and opportunities.

Your team should:
  Analyze trends and key drivers within the data
  Develop a data-driven solution or recommendation
  Clearly communicate your methodology and findings

Submission Requirements:
  5-minute recorded Zoom presentation
  Upload via the provided submission link
  Deadline: May 4, 2026 at 11:59pm ET

Showcase & Awards:
Join us for a live virtual gallery to explore projects and celebrate top teams.
Date: May 7, 2026 at 7:30pm - 8:30pm ET
